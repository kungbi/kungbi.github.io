Next.js - Component 생성

React에서 component를 생성해서 여러가지 방법으로 재사용하고 편리하게 화면을 구성할 수 있는 요소이다. Next.js에서 component를 어떻게 생성하는지 알아보았다.

컴포넌트 디렉토리

Next.js에서 컴포넌트를 생성하기 위해선 components 디렉토리가 필요하다. 최상위 디렉토리에서 components 디렉토리를 생성하자. components디렉토리에 컴포넌트를 생성하지 않으면 오류가 발생한다.
