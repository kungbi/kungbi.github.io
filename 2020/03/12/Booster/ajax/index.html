오늘은 서버와 클라이언트 사이에서 데이터 통신을 할 수 있게 하는 Ajax에 대해서 알아보려고 한다.

해당 포스팅은 커넥트재단의 edwith boostcourse - Web Programming의 강의 자료를 바탕으로 작성되었습니다.

Ajax

Ajax는 브라우저와 클라이언트와 통신을 하기 위한 기술이다. 브라우저에 데이터를 갱신할 때, 브라우저 새로코침 없이 서버로부터 데이터를 받는 것이 필요해서 만들어지게 되었다. -> 더 좋은 UX를 제공할 수 있는 기술이 되었다.

Ajax로 통신 할 때에는 XML, Plain Text, JSON 등 다양한 데이터 포멧으로 주고받을 수 있다. 보통은 사용이 편리한 JSON을 사용한다. JSON은 다음과 같이 구성되어있다.(key - value 형식)
